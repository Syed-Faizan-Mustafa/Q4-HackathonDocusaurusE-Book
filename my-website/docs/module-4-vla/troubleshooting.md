# Troubleshooting VLA

Solutions to common VLA pipeline issues.

## Whisper Issues

- Check microphone permissions
- Verify audio sample rate (16kHz)
- Try smaller model for speed

## LLM Issues

- Verify API keys
- Check rate limits
- Validate JSON output

## Latency

- Use Whisper tiny for speed
- Stream LLM responses
- Batch ROS 2 commands
