# Troubleshooting Isaac Sim

Solutions to common Isaac Sim issues.

## GPU Issues

- Verify RTX GPU with `nvidia-smi`
- Update to driver 535+
- Check VRAM usage

## Performance

- Lower ray tracing quality for testing
- Use headless mode for training
- Monitor GPU temperature
