# Isaac Sim Setup

Install and configure NVIDIA Isaac Sim for advanced robotics simulation.

## Prerequisites

- NVIDIA RTX 3070+ GPU (8GB+ VRAM)
- NVIDIA Driver 535+
- Ubuntu 22.04
- 32GB+ RAM recommended

## Installation Steps

### 1. Install Omniverse Launcher

Download from: https://www.nvidia.com/en-us/omniverse/

```bash
# Make executable and run
chmod +x omniverse-launcher-linux.AppImage
./omniverse-launcher-linux.AppImage
```

### 2. Install Isaac Sim

In Omniverse Launcher:
1. Go to Exchange tab
2. Search for "Isaac Sim"
3. Click Install

### 3. Verify Installation

Launch Isaac Sim and create a new scene.
