# Troubleshooting Hardware

Solutions to common hardware deployment issues.

## Jetson Issues

- Check power supply (65W minimum for Orin)
- Verify JetPack version
- Monitor thermal throttling

## Sensor Issues

- Check USB permissions (`sudo usermod -a -G dialout $USER`)
- Verify drivers installed
- Test with raw tools first
